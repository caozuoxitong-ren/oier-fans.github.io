# 前缀和

## **一、前缀和是什么**
前缀和是数列的前 n 项和。

## **二、前缀和怎么算**

设数列 a 共有 n 项，第 i 项为 a[i] ，1 <= i <= n . 那么，数列 a 的前 i 项和为pre[i]，则pre[i]（1<=i<=n）称做数组a的前缀和。

显然，前 1 项和 pre[1] 等于数列的第 1 项 a[1]，前 i 项和 pre[i] 等于 前 i-1 项的和 pre[i-1] 再加上第 a[i] 项。

```cpp
pre[1]=a[1]
pre[2]=a[1]+a[2]=pre[1]+a[2]
pre[3]=a[1]+a[2]+a[3]=pre[2]+a[3]
pre[4]=a[1]+a[2]+a[3]+a[4]=pre[3]+a[4]
...
pre[i]=a[1]+a[2]+...+a[i-1]+a[i]=pre[i-1]+a[i]
```

综上，求数列a的前缀和的递推公式为：

```cpp
当 i==1 时，pre[1]=a[1]
当 2<=i<=n 时，pre[i]=pre[i-1]+a[i]
```
```cpp
//代码实现：方式1
int pre[N+1]={};//pre[0]空出来置为0，方便后续使用

pre[1]=a[1];
for(int i=2;i<=n;i++) pre[i]=pre[i-1]+a[i]; 
```
```cpp
//代码实现：方式2
int pre[N+1]={};//pre[0]空出来置为0，方便后续使用

//因为pre[0]=0，所以pre[1]=pre[0]+a[1]，可以统一处理
for(int i=1;i<=n;i++) pre[i]=pre[i-1]+a[i]; 
```

## **三、用前缀和求区间和**

利用前缀和，可方便地求出数列a的在某个区间[L,R]的元素和，只需用前 R 个元素的和 pre[R] 减去前 L-1 个元素的和，就可以得到第 L 到 R 个元素的和 sum[L,R]. 

**区间和sum[L,R] = 大前缀pre[R] - 小前缀pre[L-1]**

```cpp
1 <---- pre[L-1] ---->L-1  L <---- sum[L,R] ----> R
└──────────────────────┘   └──────────────────────┘
└─────────────────────────────────────────────────┘
1 <------------------- pre[R] ------------------> R
```

```cpp
sum[L,R]
= a[L]+a[L+1]+...+a[R]
= (a[1]+a[2]+...+a[L-1]+a[L]+a[L+1]+...+a[R])-(a[1]+a[2]+...+a[L-1])
= pre[R]-pre[L-1]

即 sum[L,R]=pre[R]-pre[L-1]
```

**注意**：当L=1时，pre[L-1]=pre[0]，所以pre[]数组从pre[1]开始存储，前面空一个位置pre[0]是有用的，方便后续求解区间和；如果pre[]数组用pre[0]存储a[1]，那么当L=1时，要用if语句特判，会增加代码繁琐程度。

## **四、前缀思想的应用**

|   典型代表    |   核心操作    |   主要应用场景    |
|   :-----   |   :-----   |   :-----   |
|   **基本运算**    |
|   前缀和 (Prefix Sum) |   加法    |   快速求任意区间和 sum[L,R]=pre[R]-pre[L-1]    |
|   前缀积/后缀积 |   乘法    |   解决“除自身以外数组的乘积”等问题   |
|   前缀异或 (Prefix XOR)   |   异或    |   利用 pre[R] ^ pre[L-1] 求区间异或和 |
|   **最值与组合**  |
|   前缀最值 (Max/Min)  |   取最大/最小值   |   查询历史最值，或配合后缀最值处理“移除元素”问题。如求移除第i个元素后，剩下的元素中最大/小值是多少？  |
|   前缀 GCD / LCM  |   最大公约数/最小公倍数   |   快速求区间 GCD/LCM，或处理“移除元素”问题    |
|   **差分（前缀和的逆操作）**  |
|   差分数组 (Difference Array) |   区间增减、求前缀和  |   频繁对数组的某个区间进行统一增减操作    |
|   **统计与匹配**  |
|   前缀计数/频率   |   频数统计    |   利用计数数组快速获取任意区间内某个元素的出现次数    |
|   前缀哈希/前缀状态压缩   |   滚动哈希/位掩码 |   搭配哈希表快速寻找“和为K的子数组”或处理状态类问题   |
|   **字符串处理**  |
|   前缀函数 / KMP  |   最长公共前后缀  |   字符串单模匹配、寻找循环节  |
|   前缀树 / Trie   |   字符串插入与检索    |   搜索引擎的自动补全、拼写检查    |
|   字符串哈希  |   字符串转哈希值  |   快速比较子串是否相等（需预处理哈希值）  |
|   **高级扩展**    |
|   前缀和与二分查找    |   查找满足条件的位置  |   在有序（如正数数组）的前缀和数组上二分查找目标值    |
|   树上前缀和  |   路径权值和/异或和   |   将树上路径问题转化为两个节点到根节点的路径信息之差  |
|   高维前缀和  |   多维累加    |   处理多维空间（如图像、矩阵）的区域求和问题  |
|   后缀数组 / 后缀树   |   字符串后缀排序和压缩    |   寻找最长重复子串、全文搜索等    |
