    格式化输入函数scanf

    语法格式

    scanf 头文件<cstdio> 
    int scanf ( const char * format, ... );

    功能

    从标准输入读取格式化数据。

    从标准输入 ( stdin ) 读取数据，并根据参数格式将其存储到附加参数指向的内存地址。附加参数应指向格式字符串中相应格式说明符所指定的类型已分配的对象。

    参数

    （1）format

    一个C字符串，包含一系列字符，用于控制如何处理从流中提取字符：

    • 空白字符：该函数会读取并忽略下一个非空白字符之前遇到的所有空白字符（空白字符包括空格、换行符和制表符）。
    • 非空白字符（格式说明符 (%) 除外）：任何非空白字符（空格、换行符或制表符）或格式说明符（以%字符开头）的一部分都会导致函数从流中读取下一个字符，并将其与该非空白字符进行比较。如果匹配，则丢弃该字符，函数继续处理下一个格式字符。如果不匹配，则函数失败并返回，且流中的后续字符将不被读取。
    • 格式说明符：由初始百分号 ( % ) 组成的序列表示格式说明符，用于指定要从流中检索并存储到附加参数所指向的位置的数据类型和格式。

    scanf的 格式说明符遵循以下原型：

    %[*][width][length]specifier

    其中末尾的说明符字符（specifier）最为重要，因为它定义了要提取哪些字符、它们的解释以及其对应参数的类型：

    说明符 描述 提取的字符
    i 整数 任意位数的数字，可选择性地以符号（+或-）开头。默认为十进制数字（ 0-9），但前缀0可表示八进制数字（0-7），0x表示 十六进制数字（0-f）。带符号参数。
    d 或 u 十进制整数 任意数量的十进制数字（0-9），可以选择性地以符号（+或-）开头。d 表示有符号参数，u 表示无符号参数。
    o 八进制整数 任意数量的八进制数字（0-7），可选择性地以符号（+或-）开头。无符号参数。
    x 十六进制整数 任意数量的十六进制数字（0-9、af、AF），可选择性地以0x或0X开头，所有数字也可选择性地以符号（+或-）开头。无符号参数。
    f、e、g 浮点数 一系列十进制数字，可以包含小数点，可以前面带有符号（+或-），可以后面带有e或E字符和一个十进制整数（或支持的其他一些序列）。
    c 字符 下一个字符。如果指定的宽度不是1 ，则该函数会读取指定宽度的字符，并将它们存储在作为参数传递的数组的连续位置中。末尾不会附加空字符。
    s 字符串 任意数量的非空白字符，遇到第一个空白字符即停止。存储的序列末尾会自动添加一个终止空字符。
    p 指针地址 表示指针的字符序列。具体格式取决于系统和库的实现，但与fprintf中用于格式化 %p 的格式相同。
    [字符集] 扫描集 输入可以是任意数量括号内指定的字符。
    [^字符集] 否定扫描集 输入不能包含任何括号内指定的字符。
    n 数量 不消耗任何输入。截止到当前已经从从stdin读取的字符的数量存储在指定变量中。
    % % 一个百分号后面跟着另一个百分号，表示匹配一个百分号。

    除了n之外，任何格式说明符至少要消耗一个字符。否则匹配失败，扫描到此结束。

    格式说明符specifier还可以包含子说明符：星号( * )、宽度和长度（按此顺序），这些子说明符是可选的，并遵循以下规范：

    子规范 描述
    * 可选的起始星号表示要从流中读取数据，但要忽略该数据（即，它不会存储在参数指向的位置）。
    width 指定当前读取操作中要读取的最大字符数（可选）。
    length 可以是hh、h、l、ll、j、z、t、L中的一个（可选）。这将改变相应参数所指向的存储的预期类型（见下文）。

    下图显示了存储输入时相应参数的预期类型（包括带长度子说明符和不带长度子说明符的情况）：

    length d i u o x f e g a c s [] [^] p n
    无 int* unsigned int* float* char* void** int*
    hh signed char* unsigned char* signed char*
    h short int* unsigned short int* short int*
    l long int* unsigned long int* double* wchar_t* long int*
    ll long long int* unsigned long long int* long long int*
    j intmax_t* uintmax_t* intmax_t*
    z size_t* size_t* size_t*
    t ptrdiff_t* ptrdiff_t* ptrdiff_t*
    L long double*

    注意：hh、ll、j、z、t为C99 引入的规范和子规范。

    （2）…… （附加参数）

    根据格式字符串，scanf函数可能需要一系列附加参数，每个参数都包含一个指向已分配存储空间的指针，该存储空间用于存储提取字符的解释结果及其对应的类型。

    这些参数的数量至少应与格式说明符存储的值的数量相同。函数会忽略额外的参数。

    这些参数必须是指针：当scanf的操作结果要存储到常规变量时，常规变量的名称前应加上取地址运算符( & )。

    返回值

    成功时，该函数返回参数列表中已成功填充的项数。该计数可能与预期项数相符，也可能由于匹配失败、读取错误或到达文件末尾而少于预期项数（甚至为零） 。

    如果在读取过程中发生读取错误或到达文件末尾 ( end-of-file )，则会设置相应的指示符（ feof 或 ferror ）。如果在成功读取任何数据之前发生上述任何一种情况，则返回 EOF 。

    如果在解释宽字符时发生编码错误，则该函数会将 errno 设置为 EILSEQ 。

    兼容性

    特定的库实现可能支持额外的说明符和子说明符。

    此处列出的说明符和子说明符受最新的 C 和 C++ 标准（均发布于 2011 年）支持，但特殊说明的说明符和子说明符是 C99 引入的（仅自 C++11 及以后支持），并且可能不受旧标准库的支持。

    代码示例

    格式化串中有空白符，则scanf会读取并忽略下一个非空白字符之前遇到的所有空白字符（空白字符包括空格、换行符和制表符）。

    int a,b;
char c; 
scanf("%d%d%c",&a,&b,&c);//输入：1 2 3
printf("(%d)(%d)(%c)",a,b,c);//输出：(1)(2)( )  
//注：%c接收了2后面的空格

    int a,b;
char c; 
scanf("%d%d %c",&a,&b,&c);//输入：1 2 3
printf("(%d)(%d)(%c)",a,b,c);//输出：(1)(2)(3) 
//注：%c接收了2后面的非空字符3

    非空白字符（格式说明符 (%) 除外）：任何非空白字符（空格、换行符或制表符）或格式说明符（以%字符开头）的一部分都会导致函数从流中读取下一个字符，并将其与该非空白字符进行比较。如果匹配，则丢弃该字符，函数继续处理下一个格式字符。如果不匹配，则函数失败并返回，且流中的后续字符将不被读取。

    int a;
scanf("input:%d",&a);//输入:input:3
printf("a=%d",a); //输出：a=3 
//注："input:"对应匹配并丢弃，%d成功接收3存入a

    int a=-1;
scanf("input:%d",&a);//输入:input3
printf("a=%d",a); //输出：a=-1 
//注：冒号':'不匹配，scanf终止，a未改变依然为-1

    格式说明符：由初始百分号 ( % ) 组成的序列表示格式说明符，用于指定要从流中检索并存储到附加参数所指向的位置的数据类型和格式。

    scanf的 格式说明符遵循以下原型：%[*][width][length]specifier

    其中末尾的说明符字符（specifier）最为重要，因为它定义了要提取哪些字符、它们的解释以及其对应参数的类型。

    • %i : 有符号整型int
    int a,b,c;
scanf("%i %i %i",&a,&b,&c);//输入：10 010 0x10
printf("%i %i %i",a,b,c);//输出：10 8 16 
//注1：%i表示有符号int型整数
//注2：10是十进制，010是0开头为八进制，0x10是0x开头为16进制
    • %d：十进制有符号int型整数， %u：十进制无符号int型整数
    int a;
unsigned int b; 
scanf("%d %u",&a,&b);//输入：-1 -1 
printf("%d %u",a,b);//输出：-1 4294967295
//注1：%d表示十进制有符号int型整数，%u表示十进制无符号int型整数
//注2：-1的补码是全1，对应无符号数4294967295
    • %d：十进制有符号int型整数
    int a,b,c;
scanf("%d %d %d",&a,&b,&c);//输入：10 010 0x10 
printf("%d %d %d",a,b,c);//输出：10 10 0 
//注1：%d表示十进制有符号int型整数
//注2：%d已标明是10进制，故10按十进制为10,010按十进制为10,0x10只接受了0x前面的0
    • %o：八进制无符号int型整数
    unsigned int a,b,c,d;
scanf("%o %o %o %o",&a,&b,&c,&d);//输入：10 010 -10 -010
printf("%o %o %o %o",a,b,c,d);//输出：10 10 37777777770 37777777770
//注1：%o表示八进制无符号int型整数，对于负整数会先按有符号八进制整数解析，再转换为无符号数存储
//注2:八进制10和010对应十进制8，八进制-10和-010对应十进制-8，无符号八进制数为37777777770
    • %x：十六进制无符号int型整数
    unsigned int a,b,c,d;
scanf("%x %x %X %X",&a,&b,&c,&d);//输入：1A -1A 0x1A -0X1a
printf("%x %x %x %x\n",a,b,c,d);//输出：1a ffffffe6 1a ffffffe6
//注1：%x表示16进制无符号int型整数，对于负整数会先按16进制有符号整数解析，再转换为无符号数存储
//注2:输入时0x、0X、ABCDEF、abcdef大小写均可，输出时 %x 表示用小写输出，%X 表示用大写输出 
//注3:因%x已标明按照16进制格式读取，输入数字无论是否用0x开头均按照16进制看待
//注4:16进制1A和0x1A对应十进制26，16进制-1A和-0X1a对应十进制-26，无符号16进制数为FFFFFFE6
    • %f   %e   %g：十进制有符号float型浮点数
    float a,b,c;
scanf("%f %e %g",&a,&b,&c);//输入：123.45 1.23e4 -2E-3 
printf("%f %f %f\n",a,b,c);//输出：123.449997 12300.000000 -0.002000
printf("%e %e %e\n",a,b,c);//输出：1.234500e+002 1.230000e+004 -2.000000e-003
printf("%g %g %g\n",a,b,c);//输出：123.45 12300 -0.002
//注1：%f、%e、%g表示有符号float型浮点数 
//注2：%f表示以十进制小数形式输出（默认保留6位小数）
//注3：%e表示以指数形式（科学计数法）输出（例如 1.234560e+01）
//注4：%g则根据数值大小自动选择 %f 或 %e 中较紧凑的形式，且不输出无意义的尾随零
    • %c：字符
    char a,b;
scanf("%c%c",&a,&b);//输入：1 2 3 4 5
printf("(%c)(%c)",a,b);//输出：(1)( )
//注1：%c表示接收一个char类型字符 
//注2：scanf中"%c%c"，两个%c中间无空格，则第二个%c接收1后面的空格

    char a,b;
scanf("%c %c",&a,&b);//输入：1 2 3 4 5
printf("(%c)(%c)",a,b);//输出：(1)(2) 
//注1：%c表示接收一个char类型字符 
//注2：scanf接收字符1，因 "%c %c" 中间有空格，故跳过1后面的空白符，再接收2
    • %s：字符串
    char a[6];
scanf("%s",a);//输入：12345
printf("%s",a);//输出：12345
//注1：%s表示接收任意数量的非空白字符，遇到空白符就停止，末尾自动添加空字符'\0'
//注2：数组a要多开辟1个字节的空间用于存储尾部自动添加的空字符 '\0'
