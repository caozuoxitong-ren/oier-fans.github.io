    第 5 课 · for 循环基础语法与应用

    循环三要素 · 执行流程 · 累加器 · 累乘器 · 边界意识 · 分级题单

    三、教学重点与难点

    1. 教学重点

    • for 循环语法三要素：初始化（int i = 1）、条件（i <= n）、更新（i++）。
    • 执行流程：初始化→判断条件→循环体→更新→回判断，判断为假时退出循环。
    • 循环变量 i 的理解：从几开始、到几结束、每次加几，三个决定了循环执行几次。
    • 累加器模式：sum = 0; sum += i;——求和的标准套路。
    • 累乘器模式：product = 1; product *= i;——求阶乘的标准套路。

    2. 教学难点

    • 执行流程的理解——尤其是"更新"在"循环体"之后执行，不是先更新再执行循环体。
    • 边界判断：i < n 和 i <= n 差一次循环，写错就多打一个或少打一个。
    • 累加器初始化为 0，累乘器初始化为 1——搞反结果全错。
    • 循环变量 i 的作用域——在 for 内声明的 i，循环外不能用。
    • 数据溢出：求和时 sum 可能溢出 int，求阶乘时结果增长极快，需用 long long。

    四、为什么需要循环

    1. 前 4 课的局限

    前 4 课学的程序，每条语句只执行一次——顺序结构执行一遍，分支结构选一条路走。没有一条语句能重复执行多次。

    但如果要打印 1 到 100，难道要写 100 行 cout？要求 1 到 1000 的和，难道要写 1 + 2 + 3 + ... + 1000？

    // 不用循环：打印 1 到 5 就要写 5 行
cout << 1 << endl;
cout << 2 << endl;
cout << 3 << endl;
cout << 4 << endl;
cout << 5 << endl;

    这太笨了。如果 n 是 10000，根本写不动。

    2. 循环解决重复

    循环就是让一段代码重复执行多次。C++ 有三种循环：for、while、do-while。

    课程衔接： 三个循环中，for 最先学（本课），因为 for 的三要素（起点、终点、步长）都写在一行，最直观、最不容易出错。下一课学 while 和 do-while，再下一课学循环读入数据处理。后续还有专门课次讲解 for 与分支组合、for 循环嵌套。

    本课只学 for 循环的基本语法和基本应用。

    五、for 循环语法

    1. 语法结构

    for (初始化; 条件; 更新) {
    循环体;
}

    for 后面的括号里有三部分，用分号 ; 隔开：

    部分 作用 示例
    初始化 循环开始前执行一次，通常声明循环变量 int i = 1
    条件 每轮循环前判断，为真则执行循环体，为假则退出 i <= n
    更新 每轮循环体执行完后执行，通常改变循环变量 i++

    2. 最小示例

    // 打印 1 到 5
for (int i = 1; i <= 5; i++) {
    cout << i << endl;
}

    输出：

    1
2
3
4
5

    理解要点： i 是循环变量（习惯用 i，index 的首字母）。它从 1 开始，每次加 1，到 5 结束，循环体执行了 5 次。

    六、for 循环执行流程

    1. 执行顺序

    for 循环的执行顺序不是从左到右走一遍，而是有固定的流程：

    ① 初始化：执行 int i = 1（只执行一次）
        ↓
② 判断条件：检查 i <= n
        ↓
   为真 → ③ 执行循环体（cout << i）
        ↓
   ④ 执行更新（i++）
        ↓
   回到 ② 判断条件
        ↓
   为假 → 退出循环，执行 for 后面的语句

    关键： "更新"在"循环体"之后执行。不是先 i++ 再执行循环体，而是先执行循环体再 i++。

    2. 执行次数追踪

    以 for (int i = 1; i <= 5; i++) 为例：

    轮次 i 的值 判断 i\<=5 循环体执行 更新后 i
    第 1 轮 1 真 打印 1 2
    第 2 轮 2 真 打印 2 3
    第 3 轮 3 真 打印 3 4
    第 4 轮 4 真 打印 4 5
    第 5 轮 5 真 打印 5 6
    第 6 轮 6 假 不执行 退出循环

    循环体执行了 5 次，打印了 1 到 5。第 6 轮 i=6，6 <= 5 为假，退出循环。

    七、循环变量三要素

    for 循环的三个部分决定了循环的行为，可以归纳为三个要素：

    要素 对应部分 决定了什么 常见写法
    起点 初始化 循环变量从几开始 int i = 1
    终点 条件 循环到几结束 i <= n
    步长 更新 每次循环变量加几 i++（每次加 1）

    三个要素决定循环次数

    循环次数 = （终点 - 起点）÷ 步长，向上取整，但要结合条件判断。

    • for (int i = 1; i <= n; i++)：i 从 1 到 n，每次加 1，循环 n 次。
    • for (int i = 0; i < n; i++)：i 从 0 到 n-1，每次加 1，循环 n 次。
    • for (int i = 1; i <= n; i += 2)：i 取 1, 3, 5, ...，循环约 n/2 次。

    两种常见写法： i = 1; i <= n 和 i = 0; i < n 都循环 n 次。信奥中两种都常见，前者从 1 开始更符合数学直觉（打印 1 到 n），后者从 0 开始更符合编程习惯（数组下标从 0 开始）。本课主要用 i = 1; i <= n。

    八、基本应用 1：打印 1 到 n

    题目

    读入一个正整数 n（1 ≤ n ≤ 100），打印 1 到 n，每个数占一行。

    代码

    #include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cout << i << '\n';
    }
    return 0;
}

    输入输出示例：

    输入: 5
输出:
1
2
3
4
5

    性能意识： 用 '\n' 而非 endl，减少刷新缓冲区的开销。循环中频繁输出时，'\n' 比 endl 快。

    九、基本应用 2：打印 n 个星号

    题目

    读入一个正整数 n（1 ≤ n ≤ 100），在一行打印 n 个星号 *。

    代码

    #include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cout << '*';
    }
    cout << '\n';    // 循环结束后换行
    return 0;
}

    输入输出示例：

    输入: 5
输出: *****

    关键区别： 打印数字每个占一行（循环体内有 '\n'），打印星号都在一行（循环体内没有 '\n'，循环结束后统一换行）。循环体里有没有换行，决定了输出是一列还是一行。

    十、基本应用 3：求 1 到 n 的和（累加器）

    题目

    读入一个正整数 n（1 ≤ n ≤ 10000），求 1 + 2 + 3 + ... + n 的和。

    代码

    #include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    int sum = 0;    // 累加器初始化为 0
    for (int i = 1; i <= n; i++) {
        sum += i;   // 等价于 sum = sum + i
    }
    cout << sum << '\n';
    return 0;
}

    输入输出示例：

    输入: 100
输出: 5050

    累加器模式

    求和的核心套路叫累加器：

    1. 循环外定义一个变量 sum，初始化为 0（加法单位元是 0）。
    2. 循环体内每次执行 sum += i，把当前值累加进去。
    3. 循环结束后 sum 就是总和。
    轮次 i sum 变化
    开始 — 0
    第 1 轮 1 0 + 1 = 1
    第 2 轮 2 1 + 2 = 3
    第 3 轮 3 3 + 3 = 6
    ... ... ...
    第 n 轮 n sum + n

    数据溢出提醒： 当 n 较大时，1+2+...+n = n(n+1)/2。n = 10000 时和约 5×10⁷，在 int 范围内（约 2.1×10⁹）。但 n = 100000 时和约 5×10⁹，溢出 int，需要用 long long。拿不准就用 long long。

    十一、基本应用 4：求 n 的阶乘（累乘器）

    题目

    读入一个正整数 n（1 ≤ n ≤ 20），求 n 的阶乘 n! = 1 × 2 × 3 × ... × n。

    代码

    #include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    long long product = 1;    // 累乘器初始化为 1
    for (int i = 1; i <= n; i++) {
        product *= i;         // 等价于 product = product * i
    }
    cout << product << '\n';
    return 0;
}

    输入输出示例：

    输入: 10
输出: 3628800

    累乘器模式

    求阶乘的核心套路叫累乘器：

    1. 循环外定义一个变量 product，初始化为 1（乘法单位元是 1）。
    2. 循环体内每次执行 product *= i，把当前值乘进去。
    3. 循环结束后 product 就是乘积。

    累加器 vs 累乘器： 累加器初始化为 0，累乘器初始化为 1。搞反了结果全错——累加器初始化为 1 会多加 1，累乘器初始化为 0 会乘出来永远是 0。

    数据溢出提醒： 阶乘增长极快！12! = 479001600 已经接近 int 上限，13! 就溢出 int。20! = 2432902008176640000 接近 long long 上限，21! 就溢出 long long。所以求阶乘必须用 long long，且 n 不能太大。

    十二、复合赋值运算符

    循环中经常需要"把变量的值加上/乘以某个数再赋回给自己"，C++ 提供了简写：

    复合赋值 等价写法 含义 循环中的用途
    sum += i sum = sum + i 加后赋值 累加器
    product *= i product = product * i 乘后赋值 累乘器
    i += 2 i = i + 2 加 2 步长为 2
    i -= 1 i = i - 1 减 1 倒序（也可用 i--）
    count += 1 count = count + 1 加 1 计数器

    i++ 等价于 i += 1，等价于 i = i + 1，三种写法完全等价。i++ 最简洁，是循环中最常用的更新方式。

    十三、循环边界意识：< 与 <=

    1. 差一次循环

    // 写法 A：i <= n，i 取 1,2,...,n，循环 n 次
for (int i = 1; i <= n; i++) { ... }

// 写法 B：i < n，i 取 1,2,...,n-1，循环 n-1 次
for (int i = 1; i < n; i++) { ... }

    <= 和 < 差一次循环。写错了就多打一个或少打一个。

    2. 怎么选

    条件写法 i 的取值范围 循环次数 适合场景
    i <= n 1 到 n n 次 打印 1 到 n、求 1 到 n 的和
    i < n 1 到 n-1 n-1 次 从 0 开始的写法：i = 0; i < n 循环 n 次

    建议： 从 1 开始的循环用 i <= n（包含 n），从 0 开始的循环用 i < n（不包含 n）。两种写法都循环 n 次，但含义不同。写完回头检查边界——这是循环题最容易错的地方。

    十四、for 循环的变体

    1. 倒序循环

    把更新改成 i--，循环变量递减：

    // 打印 n 到 1
for (int i = n; i >= 1; i--) {
    cout << i << '\n';
}

    输入 5，输出 5 4 3 2 1（每个占一行）。

    倒序循环：起点大、终点小、步长为负（i--），条件用 >=。

    2. 步长为 2

    // 打印 1 到 n 中的奇数
for (int i = 1; i <= n; i += 2) {
    cout << i << ' ';
}

    输入 10，输出 1 3 5 7 9 。

    步长不一定是 1。i += 2 每次加 2，i += 5 每次加 5，根据需要选择。

    十五、常见错误与注意事项

    编号 错误 后果 改正
    E1 条件写成 i < n 而非 i <= n 少循环一次，少打印/少加一个数 检查边界，从 1 开始用 <=
    E2 累加器初始化为 1 而非 0 结果多 1 累加器初始化为 0
    E3 累乘器初始化为 0 而非 1 结果永远是 0 累乘器初始化为 1
    E4 求阶乘用 int 而非 long long 13! 溢出 int，结果错误 阶乘用 long long
    E5 循环体后多写了分号 循环体为空，只执行更新不执行预期语句 删掉 for 行末的分号
    E6 更新写成 i+1 而非 i++ 或 i+=1 i 永远不变，死循环 用 i++ 或 i += 1
    E7 条件写成 i = n（赋值）而非 i <= n = 是赋值，条件恒为 n（非 0 即真），死循环 条件用 <= 或 <，不能用 =
    E8 循环变量在循环外使用 for 内声明的 i，循环外不可见，编译报错 在循环外声明 i，或不在循环外用 i
    E9 忘记写更新部分 i 永远不变，死循环 三要素缺一不可
    E10 cout << i << endl 写成 cout << i 不换行 数字全挤在一行 需要每行一个时加 '\n'

    死循环

    如果条件永远为真，循环永远不会结束，这叫死循环。最常见的原因是忘记写更新或更新写错：

    // 死循环：忘记 i++，i 永远是 1，1 <= n 永远为真
for (int i = 1; i <= n; ) {
    cout << i;
}

    调试建议： 程序卡住不动，很可能是死循环。检查 for 的三要素是否齐全、更新是否正确改变循环变量。

    for 行末多分号

    // 错误：for 行末多了分号，循环体为空
for (int i = 1; i <= n; i++); {
    cout << i;    // 这不在循环体内！只执行一次
}

    for 后面的 {} 前面多了个 ;，{} 变成了普通代码块而非循环体。循环只执行 i++ 更新，不执行 cout。

    十六、方法、套路与策略小结

    1. for 循环三步套路

    • 第 1 步 确定三要素： 起点从几开始、终点到几结束、步长每次加几。
    • 第 2 步 写循环体： 每轮要做什么（打印、累加、累乘）。
    • 第 3 步 检查边界： 条件用 < 还是 <=，循环执行了几次。

    2. 累加器与累乘器决策

    任务 模式 初始化 循环体 数据类型
    求和 累加器 sum = 0 sum += i n 大时用 long long
    求阶乘 累乘器 product = 1 product *= i 必须用 long long
    求平方和 累加器 sum = 0 sum += i * i 用 long long
    打印数字 直接输出 — cout << i 用 int

    3. 循环次数决策

    写法 i 的取值 循环次数
    i = 1; i <= n; i++ 1, 2, ..., n n 次
    i = 0; i < n; i++ 0, 1, ..., n-1 n 次
    i = 1; i < n; i++ 1, 2, ..., n-1 n-1 次
    i = n; i >= 1; i-- n, n-1, ..., 1 n 次

    4. 信奥策略意识

    • 三要素齐全： 初始化、条件、更新，缺一不可。写完 for 先检查三部分是否都有。
    • 边界必测： 测 n=1（最小值）和 n=较大值，检查循环是否执行了正确次数。
    • 累加器初始化 0，累乘器初始化 1： 这是铁律，搞反结果全错。
    • 数据溢出意识： 求和 n 大时用 long long，求阶乘必须用 long long。拿不准就用 long long。
    • 性能意识： 循环内频繁输出用 '\n' 而非 endl。
    • 样例验证： 手动追踪循环变量 i 的变化，确认循环执行次数和输出结果。

    附录 · 本课代码模板速查

    模板 1：for 循环万能框架

    #include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        // 循环体：每轮做什么
    }
    return 0;
}

    三要素：int i = 1（起点）、i <= n（终点）、i++（步长）。改这三部分就能适应不同需求。

    模板 2：累加器模板（求和）

    int n;
cin >> n;
int sum = 0;            // 累加器初始化为 0
for (int i = 1; i <= n; i++) {
    sum += i;           // 累加
}
cout << sum << '\n';

    求和的标准套路。改 sum += i 为 sum += i * i 就能求平方和。

    模板 3：累乘器模板（求阶乘）

    int n;
cin >> n;
long long product = 1;  // 累乘器初始化为 1，必须用 long long
for (int i = 1; i <= n; i++) {
    product *= i;       // 累乘
}
cout << product << '\n';

    求阶乘的标准套路。必须用 long long，13! 就溢出 int。

    模板 4：打印一行字符

    int n;
cin >> n;
for (int i = 1; i <= n; i++) {
    cout << '*';        // 循环体内不换行
}
cout << '\n';           // 循环结束后换行

    打印 n 个字符在一行。改 '*' 为其他字符即可。

    模板 5：倒序循环

    for (int i = n; i >= 1; i--) {
    cout << i << '\n';
}

    倒序：起点大、终点小、步长为负（i--）、条件用 >=。

    模板 6：步长为 2 的循环

    // 打印 1 到 n 中的奇数
for (int i = 1; i <= n; i += 2) {
    cout << i << ' ';
}
cout << '\n';

    步长不一定是 1。i += 2 每次加 2，根据需要调整。
