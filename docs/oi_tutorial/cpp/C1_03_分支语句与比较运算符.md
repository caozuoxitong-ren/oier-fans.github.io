## 三、教学重点与难点

### 1\. 教学重点

-   6 个比较运算符 `==`、`!=`、`<`、`>`、`<=`、`>=`，尤其 `==`（判断相等）与 `=`（赋值）的区别。
-   if 三种形式：单分支 `if`、双分支 `if-else`、多分支 `if-else if-else`。
-   花括号 `{ }` 的使用：单条语句可省，多条语句必须加；建议初学者一律加。

### 2\. 教学难点

-   `==` 与 `=` 的混淆——这是分支结构第一杀手，`if (a = 0)` 不报错但逻辑全错。
-   if 后面误加分号 `if (a > 0);`——空语句陷阱，条件成立却什么都不做。
-   比较运算符不能连写：`a < b < c` 是错误的，本课阶段先用"分开判断"避免（第 4 课学 `&&` 后用 `a < b && b < c`）。
-   `else if` 的隐含条件理解：走到 `else if` 说明前面的条件都不成立。

## 四、为什么需要分支

### 1\. 第二课的局限

第二课的程序都是"从头到尾一条路走到底"——每条语句都执行，没有跳过、没有选择。这叫**顺序结构**。

但现实问题常常需要"看情况"：

-   成绩 ≥ 60 分，及格；否则不及格。
-   一个数是正数、负数还是零？
-   三个数谁最大？

这些都需要程序**根据条件做判断、走不同的路**——这就是**分支结构**。

### 2\. 分支带来的改变

```cpp
int score;
cin >> score;
if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格" << endl;
}

```

输入 `80` 输出"及格"，输入 `50` 输出"不及格"——同一份代码，根据数据走不同分支。这就是分支的价值。

### 3\. 本课的边界

本课只讲**单个条件**的判断——比如"成绩 ≥ 60""n 是偶数"。如果一个判断需要多个条件组合（比如"闰年要同时满足能被 4 整除且不能被 100 整除"），那是第 4 课的内容。先把单条件分支学扎实，再学多条件组合。

## 五、比较运算符

### 1\. 六个比较运算符

比较运算符用来"比较两个值的大小或是否相等"，结果是 `bool` 类型：**成立为 `true`（等于 1），不成立为 `false`（等于 0）**。

| 运算符 | 含义 | 示例 | 结果 |
| --- | --- | --- | --- |
| `==` | 等于 | `3 == 3` | `true`(1) |
| `!=` | 不等于 | `3 != 4` | `true`(1) |
| `<` | 小于 | `3 < 4` | `true`(1) |
| `>` | 大于 | `3 > 4` | `false`(0) |
| `<=` | 小于等于 | `4 <= 4` | `true`(1) |
| `>=` | 大于等于 | `3 >= 4` | `false`(0) |

### 2\. 最关键的区分：`==` 与 `=`

| 符号 | 名称 | 作用 | 示例 |
| --- | --- | --- | --- |
| `=` | 赋值运算符 | 把右边的值放进左边的变量 | `a = 5;` |
| `==` | 等于运算符 | 判断两边是否相等，结果是真/假 | `a == 5` |

> **第一杀手：** `if (a == 0)` 是"判断 a 是否等于 0"；`if (a = 0)` 是"把 0 赋值给 a，再判断 a（此时是 0，即假）"。后者不报错，但条件永远不成立，逻辑全错。**判断相等一定写两个等号。**

### 3\. 比较的结果可以直接输出

```cpp
cout << (3 < 4);      // 输出 1（true）
cout << (3 > 4);      // 输出 0（false）
cout << (5 == 5);     // 输出 1

```

### 4\. 常见用法

```cpp
n % 2 == 0        // n 是偶数
n % 2 == 1        // n 是奇数
score >= 60       // 成绩及格
a != 0            // a 不等于 0

```

## 六、比较运算符的优先级与结果

### 1\. 优先级低于算术运算符

比较运算符的优先级**低于**算术运算符，所以 `a + b > c` 等价于 `(a + b) > c`，先算加法再比较，符合数学习惯。

```cpp
int a = 3, b = 4, c = 5;
cout << (a + b > c);     // 7 > 5，输出 1（true）
cout << (a * b >= c);    // 12 >= 5，输出 1（true）

```

### 2\. 比较运算符不能连写

```cpp
// 错误写法
if (a < b < c) { ... }   // 先算 a<b 得 0 或 1，再和 c 比，逻辑全错

// 正确写法（本课阶段：分两次判断）
if (a < b) {
    if (b < c) {         // 嵌套 if，第 4 课详细讲
        cout << "a < b < c" << endl;
    }
}

```

> **第 4 课预告：** 学了逻辑运算符 `&&` 后，可以写成 `if (a < b && b < c)`，更简洁。本课阶段先用嵌套 if 或分步判断。

### 3\. 赋值运算符 `=` 优先级更低

`=` 的优先级低于比较运算符，所以 `a = b > c` 等价于 `a = (b > c)`——先把 `b > c` 的真假值赋给 a。这通常不是你想要的，**判断条件里不要混用 `=`**。

## 七、if 单分支语句

### 1\. 语法

```cpp
if (条件) {
    // 条件成立时执行的语句
}

```

**执行流程：** 判断条件——成立（真）就执行花括号里的语句，不成立（假）就跳过，什么都不做。

### 2\. 示例：判断正数

```cpp
int n;
cin >> n;
if (n > 0) {
    cout << n << " 是正数" << endl;
}

```

输入 `5` 输出"5 是正数"；输入 `-3` 什么都不输出。

### 3\. 花括号规则

-   花括号里只有**一条语句**时，花括号可省：`if (n > 0) cout << "正数";`
-   但**强烈建议初学者一律加花括号**，避免后续添加语句时漏加导致 bug。

```cpp
// 危险写法（容易出错）
if (n > 0)
    cout << "正数" << endl;
    cout << "结束" << endl;   // 这行不在 if 内！缩进骗人，它总会执行

// 安全写法
if (n > 0) {
    cout << "正数" << endl;
    cout << "结束" << endl;   // 两行都在 if 内
}

```

> **规范意识：** 一律加花括号，哪怕只有一条语句。这是杜绝"漏括号"bug 最简单有效的办法。

## 八、if-else 双分支语句

### 1\. 语法

```cpp
if (条件) {
    // 条件成立时执行
} else {
    // 条件不成立时执行
}

```

**执行流程：** 二选一。条件成立走 if 分支，不成立走 else 分支，两条路必走一条。

### 2\. 示例：判断及格

```cpp
int score;
cin >> score;
if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格" << endl;
}

```

### 3\. 示例：判断奇偶

```cpp
int n;
cin >> n;
if (n % 2 == 0) {
    cout << "偶数" << endl;
} else {
    cout << "奇数" << endl;
}

```

### 4\. 示例：两数较大值

```cpp
int a, b;
cin >> a >> b;
if (a >= b) {
    cout << a << endl;
} else {
    cout << b << endl;
}

```

## 九、if-else if-else 多分支语句

### 1\. 语法

```cpp
if (条件1) {
    // 条件1 成立
} else if (条件2) {
    // 条件1 不成立，条件2 成立
} else if (条件3) {
    // 条件1、2 都不成立，条件3 成立
} else {
    // 以上都不成立
}

```

**执行流程：** 从上往下依次判断，**一旦某个条件成立，执行对应分支后跳出整个 if 结构**，不再往下判断。都不成立则走 else。

### 2\. 示例：成绩等级

```cpp
int score;
cin >> score;
if (score >= 90) {
    cout << "优秀" << endl;
} else if (score >= 80) {
    cout << "良好" << endl;
} else if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格" << endl;
}

```

> **关键理解：** 上面的条件已经隐含了"前面不成立"。比如 `else if (score >= 80)` 实际意思是"score \< 90 且 score ≥ 80"。因为能走到这里说明 `score >= 90` 不成立，不用再写 `score < 90`。

### 3\. 示例：判断正负零

```cpp
int n;
cin >> n;
if (n > 0) {
    cout << "正数" << endl;
} else if (n < 0) {
    cout << "负数" << endl;
} else {
    cout << "零" << endl;
}

```

### 4\. else 可以省略

如果不需要"以上都不成立"的兜底分支，最后的 `else` 可以省略：

```cpp
if (n > 0) {
    cout << "正数" << endl;
} else if (n < 0) {
    cout << "负数" << endl;
}
// n == 0 时什么都不输出

```

### 5\. 注意：else if 不是独立关键字

`else if` 本质是 `else` 后面跟一个完整的 `if`。下面两种写法等价：

```cpp
// 写法一（推荐，简洁）
if (a > 0) { ... }
else if (a < 0) { ... }
else { ... }

// 写法二（展开，等价）
if (a > 0) { ... }
else {
    if (a < 0) { ... }
    else { ... }
}

```

## 十、常见错误与注意事项

| 编号 | 错误 | 后果 | 改正 |
| --- | --- | --- | --- |
| E1 | `=` 写成判断（`if (a = 0)`） | 不报错，条件永远假（或永远真），逻辑全错 | 判断相等用 `==` |
| E2 | if 后加分号（`if (a > 0);`） | 空语句，条件成立却什么都不做 | 删掉分号 |
| E3 | 漏花括号导致多条语句只有第一条在 if 内 | 后续语句总会执行 | 一律加花括号 |
| E4 | `else if` 写成 `if`（漏 else） | 多个 if 独立判断，可能都执行 | 多分支用 `else if` 串联 |
| E5 | 条件写反（`>` 写成 `<`） | 逻辑反了 | 仔细核对题意 |
| E6 | 比较运算符连写（`a < b < c`） | 先算 `a<b` 得 0/1，再和 c 比，逻辑错 | 分步判断或用嵌套 if |
| E7 | `==` 写成单个 `=` 在条件里 | 变成赋值，条件永远假/真 | 判断用两个等号 |
| E8 | else 后面加分号（`} else ; {`） | else 分支变空语句 | else 后直接跟花括号 |

### if 后加分号的陷阱详解

```cpp
// 错误
if (a > 0); {
    cout << "正数" << endl;
}

```

这个分号让 if 后面跟了一条**空语句**（什么都不做）。花括号里的 `cout` 不在 if 内，**无论条件是否成立都会执行**。编译不报错，但逻辑全错。

> **口诀：** if、else 后面**不加分号**，直接跟花括号。

### 输入法提醒

-   写代码一律英文输入法。`==`、`!=`、`<=`、`>=` 都是英文符号，中文符号会报错。

## 十一、方法、套路与策略小结

### 1\. 分支结构三步套路

-   **第1步 读入：** 读入需要判断的数据。
-   **第2步 判断：** 用比较运算符写出条件，组织成 if-else 结构。
-   **第3步 输出：** 每个分支内处理对应情况。

### 2\. if 形式选择决策

| 场景 | 用法 |
| --- | --- |
| 满足条件做事，不满足不管 | 单分支 `if` |
| 二选一 | `if-else` |
| 多选一（按范围或等级） | `if-else if-else` |

### 3\. 条件书写要点

-   判断相等用 `==`，不是 `=`。
-   范围判断本课阶段用嵌套 if 分步判断（第 4 课学 `&&` 后可合并）。
-   比较运算符不能连写 `a < b < c`。

### 4\. 信奥策略意识

-   **判断相等用 `==`：** 写完条件回头检查一遍，这是分支题第一易错点。
-   **一律加花括号：** 哪怕只有一条语句也加，杜绝漏括号 bug。
-   **样例验证：** 分支题要测**每个分支**——及格/不及格、正/负/零都要测到，不能只测一种情况。
-   **边界必测：** `>= 60` 的题，测 59、60、61 三个边界值。

## 附录 · 本课代码模板速查

### 模板 1：if 三种形式

```cpp
// 单分支
if (条件) {
    // 成立时执行
}

// 双分支
if (条件) {
    // 成立时执行
} else {
    // 不成立时执行
}

// 多分支
if (条件1) {
    // 条件1 成立
} else if (条件2) {
    // 条件2 成立
} else {
    // 都不成立
}

```

### 模板 2：分支结构万能框架（读入→判断→输出）

```cpp
#include <iostream>
using namespace std;
int main(){
    int n;
    cin >> n;                    // 1. 读入
    if (n % 2 == 0) {            // 2. 判断
        cout << "偶数" << endl;  // 3. 输出
    } else {
        cout << "奇数" << endl;
    }
    return 0;
}

```

### 模板 3：比较运算符速记

```cpp
a == b   // 等于（判断，两个等号）
a != b   // 不等于
a < b    // 小于
a > b    // 大于
a <= b   // 小于等于
a >= b   // 大于等于

// 常见用法
n % 2 == 0     // 偶数
n % 2 == 1     // 奇数
score >= 60    // 及格
a != 0         // 不为零

```

### 模板 4：三数排序（比较交换法）

```cpp
int a, b, c;
cin >> a >> b >> c;
int t;
if (a > b) { t = a; a = b; b = t; }   // 保证 a <= b
if (b > c) { t = b; b = c; c = t; }   // 保证 b <= c
if (a > b) { t = a; a = b; b = t; }   // 再保证 a <= b
cout << a << " " << b << " " << c << endl;  // 从小到大

```

> **交换变量三步法：** 借助临时变量 `t`，`t=a; a=b; b=t;` 完成 a、b 互换。这是 P5715（三位数排序）的核心技巧。每次比较都是单个条件，不需要逻辑运算符。

### 模板 5：三数最大值（逐步更新法）

```cpp
int a, b, c;
cin >> a >> b >> c;
int mx = a;                  // 先假设 a 最大
if (b > mx) { mx = b; }     // b 比 mx 大就更新
if (c > mx) { mx = c; }     // c 比 mx 大就更新
cout << mx << endl;

```

> 每次比较都是单个条件，逐步更新最大值。这种方法在第 4 课学逻辑运算符后依然适用，是通用的求最大值套路。
