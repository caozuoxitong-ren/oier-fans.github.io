# 素数的判定

## 什么是素数？

素数，又称为质数，指在一个大于 $1$ 的自然数中，除了 $1$ 和此整数自身外，无法被其他自然数整除的数（也可以定义为只有 $1$ 和本身两个因数的数）。

比 $1$ 大但不是素数的数成为合数。 $1$ 和 $0$ 既非素数也非合数。

素数在数论中有着很重要的地位。

## 素数的判定

### 朴素枚举法

思路：要判断 $N$ 是否为素数，只需检查从 $2$ 到 $N-1$ 的每一个数是否能整除 $N$ 即可。若存在能整除 $N$ 的数，则说明 $N$ 不是素数；若都不能整除 $N$ ，则说明 $N$ 为素数。时间复杂度为 $O(n)$ .

代码实现：
```cpp
bool isPrime(int n){
	if(n<2) return false;
	for(int i=2;i<=n-1;i++){
		if(n%i==0) return false;
	}
	return true;
} 
```

### 平方根优化法

思路：整数 $N$ 的因子是成对出现的：若 $1$ 是 $N$ 的因子，则 $N/1$ 必是 $N$ 的因子；若 $2$ 是 $N$ 的因子，则 $N/2$ 必是 $N$ 的因子；若 $3$ 是 $N$ 的因子，则 $N/3$ 必是 $N$ 的因子；... ...；若 $N$ 是 $N$ 的因子，则 $N/N$ 必是 $N$ 的因子。

可能的**因子对**：<br>
$1$　　　　$N/1$ <br>
$2$　　　　$N/2$ <br>
$3$　　　　$N/3$ <br>
$4$　　　　$N/4$ <br>
$5$　　　　$N/5$ <br>
$...$　　　$...$ <br>
$\sqrt{N}$　　$\sqrt{N}$ <br>
$...$　　　$...$ <br>
$N/5$　　　$5$ <br>
$N/4$　　　$4$ <br>
$N/3$　　　$3$ <br>
$N/2$　　　$2$ <br>
$N/1$　　　$1$ <br>

可以发现：

1. 以$\sqrt{N}$为界，前半部分的因子对和后半部分的因子对是左右翻转对称的，因此只需考察$\sqrt{N}$之前所有的因子对即可。
2. 对于$\sqrt{N}$及之前的因子对，若左侧因子 $i$ 能整除 $N$ ，则右侧因子 $N/i$ 必能整除 $N$ ，因此只需考察前半部分的左侧因子即可。

所以，只需检查区间 $[2,\sqrt{N}]$ 内的每个数字能否整除 $N$，即可判断 $N$ 是否为素数。时间复杂度为 $O(\sqrt{N})$.

代码实现：
```cpp
bool isPrime(int n){
	if(n<2) return false;
	for(int i=2;i<=n/i;i++){
		if(n%i==0) return false;
	}
	return true;
}
```
<div style="text-align: justify;">
需要注意，上述代码中 for 循环判定条件的写法是 i<=N/i，若写为 i*i<=N 则 i*i 可能会溢出，从而导致结果错误。32 位的 int 类型有符号数最大是 $2147483647$ ，$46340 \le \sqrt{2147483647} \le 46341$ 。因此，当要判断的数为 $N=2147483647$ 时，枚举到 $i=46340$ 时，因为 $46340^2 = 2147395600$ 不超过 int 最大值，可以正常运行；进一步 $i=46341$ 时，$46341^2 = 2147488281 \gt 2147483647$ 超过了 int 最大值，就会溢出导致程序错误。
</div>
